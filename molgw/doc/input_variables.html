Input variable list



scf postscf alpha_mixing beta_mixing gamma_mixing basis auxil_basis basis_path gaussian_type no_4center

  • scf

    Default:

    Contains the self-consistent scheme name. Try LDA, PBE, HSE06, or HF for instance

  • postscf

    Default:

    Contains the post-processing scheme name. TD stands for TD-DFT BSE stands for Bethe-Salpeter GW stands for perturbative G0W0 GnW0 stands for GW with eigenvalue self-consistentcy on G GnWn stands for GW with eigenvalue self-consistentcy on both G and W MP2 stands for guess what.

  • alpha_mixing

    Default: 0.25

    Only works for Range-Separated hybrid functionals scf='rsh' Sets the amount of range-independent exact-exchange

  • beta_mixing

    Default: 0.0

    Only works for Range-Separated hybrid functionals scf='rsh' Sets the amount of long-range exact-exchange

  • gamma_mixing

    Default: 1000000.0

    Only works for Range-Separated hybrid functionals scf='rsh' Sets the separation between long-range and short-range. It is input in bohr^-1.

  • basis

    Default:

    Sets the basis set For Pople sets, use 6-31G for instance or 6-31pGs, where p stands for + and s for *. For Dunning sets, use aug-cc-pVTZ for instance. Note that Pople sets are to be used with gaussian_type='cart' One may use ones own basis sets provided that the files are labeled X_mybasisset where X is the element.

  • auxil_basis

    Default:

    Sets the auxiliary basis set. For instance, cc-pVDZ-RI for a Weigend basis set. If present, the auxiliary basis will be used for postscf calculations (TD-DFT, BSE, or GW) If specifically requested with no_4center, the auxiliary basis can be used for scf cycles too.

  • basis_path

    Default: ./

    Sets the path pointing to the basis functions files.

  • gaussian_type

    Default: pure

    Asks for pure or spherical Gaussian type orbitals with 'pure' or forCartesian Gaussian orbital with 'cart'.

  • no_4center

    Default: no

    If switched on, the auxiliary basis set is used in both SCF cycles and in post-scf methods.This avoids the calculation and the storage of the 4-center Coulomb integrals.
